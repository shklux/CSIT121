Reflection from Lab 05

In this lab, I learned how to create a responsive web design that adapts to different screen sizes. I used CSS Flexbox to make the layout responsive, ensuring that the website is functional on both desktop and mobile devices. Additionally, I learned about best design practices, including the use of media queries for layout adjustments and improving accessibility across devices.

Best Design Practices:

  • Ensure layouts are responsive to different screen sizes
  • Use semantic HTML for better accessibility
  • Prioritize user experience on both mobile and desktop devices
